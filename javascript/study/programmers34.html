코팅테스트 연습 > 연습문제 > 부족한 금액 계산하기

❓ 문제 설명

새로 생긴 놀이기구는 인기가 매우 많아 줄이 끊이질 않습니다. 이 놀이기구의 원래 이용료는 price원 인데, 놀이기구를 N 번 째 이용한다면 원래 이용료의 N배를 받기로 하였습니다. 즉, 처음 이용료가 100이었다면 2번째에는 200, 3번째에는 300으로 요금이 인상됩니다. 놀이기구를 count번 타게 되면 현재 자신이 가지고 있는 금액에서 얼마가 모자라는지를 return 하도록 solution 함수를 완성하세요. 단, 금액이 부족하지 않으면 0을 return 하세요.

❗ 제한사항

놀이기구의 이용료 price : 1 ≤ price ≤ 2,500, price는 자연수 처음 가지고 있던 금액 money : 1 ≤ money ≤ 1,000,000,000, money는 자연수 놀이기구의 이용 횟수 count : 1 ≤ count ≤ 2,500, count는 자연수

🍖 입출력 예

price money count result
3 20 4 10

🍙 입출력 예 설명

입출력 예 #1

이용금액이 3인 놀이기구를 4번 타고 싶은 고객이 현재 가진 금액이 20이라면, 총 필요한 놀이기구의 이용 금액은 30 (= 3+6+9+12) 이 되어 10만큼 부족하므로 10을 return 합니다.

⏳ 문제 풀이

function solution(price, money, count) {
    let result = "";
    let sumPrice = 0;
    
    for(let i = 1; i <= count; i++) {
        sumPrice += price * i;   
    }
    if(sumPrice <= money) {
        result = 0;
    } else if(sumPrice > money) {
        result = sumPrice - money;
    }
    return result;
}

먼저 누적되는 합을 구해 담을 수 있는 sumPrice를 선언해줍니다.
for문을 활용하여 count만큼 순회하며 price에 i를 곱한 값을 먼저 선언 해 둔 sumPrice에 담습니다.
이 때 fot문에서 선언하는 i값은 1부터 시작해야 count값만큼만 순회할 수 있습니다.
이후 for문 밖에서 if문을 활용해 sumPrice가 money보다 작거나 같은 경우에는 result에 0을 담고
큰 경우는 else if로 조건을 넣어 result에 sumPrice에서 money를 뺀 값을 담습니다.
if문 밖에서 result를 return하면 끝입니다.

👀 다른 사람의 풀이

function solution(price, money, count) {
    let answer = 0;

    for (let i = 1; i <= count; i++) {
        answer += price * i;
    }

    return answer > money ? answer - money : 0;
}
